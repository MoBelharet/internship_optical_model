To compile the FABM-PISCES model:
--> Go to the project home 
--> ./build_fabm_config.sh

To compile The NEMO-FABM-PISCES 
--> Go to NEMO4.0-FABM
--> ./compile clean
--> ./compile

To run a simulation:
--> Go to run_nemo_fabm_pisces
--> copy the executable file nemo.exe from the compilation folder 
--> bash run_job.sh
